# TP DevOps

Ce projet illustre les bases de l'automatisation du développement avec **Node.js**, **Docker** et **GitHub Actions**.

## Fonctionnalités

- Une fonction simple `sum(a, b)` avec son test automatisé.
- Tests exécutés avec **Jest**.
- Pipeline CI GitHub Actions pour :
  - Installer les dépendances
  - Construire l'image Docker
  - (Possibilité d'ajouter les tests à l'avenir)

## Installation

```bash
npm install
